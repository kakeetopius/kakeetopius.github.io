I’m a systems and network engineer with a strong focus on Linux, networking, and cloud infrastructure. I primarily work with Go and Python to build practical and reliable tools, while occasionally using C, Java, and Rust for low-level or specialized tasks. I enjoy combining low-level systems work, networking, and cloud technologies to develop solutions that are both efficient and robust.

## What I Do

I can work across different fields with expertise in:

- **Network Engineering & Automation** — Designing and managing scalable networks; configuring devices; automating tasks and developing networking utilities in Go and Python.
- **Network Security** — Securing networks, implementing best practices, monitoring, and building tools to enhance network security.
- **Linux Systems & Security Engineering** — Proficient in Linux administration, hardening servers, automating tasks, and implementing security best practices.
- **Cloud & DevOps** — AWS, Docker, Kubernetes, CI/CD pipelines
- **Open Source** — Contributing to projects and building developer tools
- **Backend Development** — Scalable APIs and microservices in Go.
